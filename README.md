#### Development

*Front-end (Vue.js)*

```
cd ./lab3/client
npm install
npm run serve
```

Visit http://localhost:8080 to test and debug your front-end application.

----

*Back-end (Node.js)*

```
cd ./lab3/server
npm install
npm run start
```

Visit http://localhost:3000 to see if your server loads the front-end application.
